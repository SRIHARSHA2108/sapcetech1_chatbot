# SpaceTech Chatbot

A simple chatbot web application built using Python and Flask.  
This project provides a basic frontend interface where users can interact with a chatbot.

## 🚀 Project Features

- Simple chatbot user interface
- Python Flask backend
- HTML, CSS, and JavaScript frontend
- Template-based web pages
- Static files support
- Easy to run locally

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask

### Database
- MongoDB
